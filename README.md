# 🧠 8086 Assembly Language Lab

A curated repository of my **Assembly Language (8086)** lab work, demonstrating core low-level programming concepts, algorithm design, and direct interaction with hardware-level instructions.

---

## 📌 About

This repository reflects hands-on implementation of fundamental programming techniques using the **Intel 8086 architecture**.
Each task is carefully written to ensure clarity, correctness, and efficient use of registers and memory.

## ⚙️ Key Concepts Covered

* Register-level data manipulation
* Conditional logic & branching
* Loop construction in Assembly
* Memory addressing techniques
* Array processing
* Sorting algorithms (Bubble Sort)
* Basic optimization (flag-based execution)

---

## 🚀 Highlight Implementation

### 🔹 Bubble Sort (Task 4)

* Sorts a fixed array of 10 elements
* Uses **byte-sized values (DB: 0–255)**
* Includes **swap flag optimization** for improved efficiency
* Designed specifically for **8086 instruction set**
